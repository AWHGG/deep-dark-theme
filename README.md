# Deep Dark Theme

一款极致深色的 VS Code 主题，采用 Tokyo Night Storm 风格的语法高亮配色。

## ✨ 特点

- **超深背景色** - 主背景 `#202124`，编辑区 `#27282b`，极致护眼
- **统一低对比度** - 所有 UI 元素亮度协调一致，减少视觉疲劳
- **Tokyo Night 配色** - 采用流行的 Tokyo Night Storm 语法高亮方案
- **无边框设计** - 去除多余边框线条，界面简洁干净
- **完整 UI 覆盖** - 覆盖侧边栏、状态栏、面板、终端等所有区域

## 🎨 颜色方案

| 元素 | 颜色 |
|------|------|
| 主背景 | `#202124` |
| 编辑区背景 | `#27282b` |
| 前景文字 | `#bbbbbb` |
| 关键字 | `#9d7cd8` |
| 字符串 | `#9ece6a` |
| 函数 | `#7aa2f7` |
| 常量 | `#ff9e64` |
| 类型 | `#2ac3de` |

## 📦 安装

### 从 VS Code 商店安装
1. 打开 VS Code
2. 按 `Ctrl+Shift+X` 打开扩展面板
3. 搜索 "Deep Dark Theme"
4. 点击安装

### 手动安装
1. 下载 `.vsix` 文件
2. 按 `Ctrl+Shift+P`，输入 "Install from VSIX"
3. 选择下载的文件

## 🔧 激活主题

1. 按 `Ctrl+K Ctrl+T` 打开主题选择器
2. 选择 "Deep Dark"

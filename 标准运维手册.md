# 标准运维手册

## 1. 项目基础信息
- **项目名称**: Deep Dark Theme
- **项目类型**: VS Code Extension (Color Theme)
- **代码仓库**: https://github.com/geniu/deep-dark-theme
- **当前版本**: v0.4.2

## 2. 环境配置
### 本地开发环境
- **系统**: Windows / macOS / Linux
- **依赖**: 
  - Node.js (推荐 v16+)
  - VS Code
  - `vsce` (Visual Studio Code Extensions CLI)
    ```bash
    npm install -g @vscode/vsce
    ```

### 调试方法
1. 在 VS Code 中打开项目根目录。
2. 按 `F5` 启动调试窗口（Extension Development Host）。
3. 在新窗口中按 `Ctrl+K Ctrl+T` 切换到 "Deep Dark" 主题进行预览。

## 3. 部署与发布流程
### 发布到 VS Code Marketplace
> **注意**: 发布前请确保已更新 `package.json` 中的版本号。

1. **打包与验证**:
   ```bash
   vsce package
   ```
   此命令会生成 `.vsix` 文件，可用于本地测试安装。

2. **发布**:
   ```bash
   vsce publish
   ```
   *需要 Personal Access Token (PAT)*。

3. **Git 同步**:
   - 提交代码并推送到远程仓库。
   - 打上版本 Tag (可选)。
